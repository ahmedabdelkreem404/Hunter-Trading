**Project Title:**
Hunter Trading – Premium Trading Education & Affiliate Platform

---

**Project Overview:**
Build a high-end, modern, and conversion-focused website for a personal trading coach brand called “Hunter Trading”. The platform should combine education, community, and affiliate monetization into a seamless user experience. The main goal is to convert visitors into Telegram subscribers and affiliate referrals (broker sign-ups).

---

**Tech Stack Requirements (STRICT):**

* Backend: Core PHP (NO frameworks like Laravel)
* Database: MySQL
* Frontend: React.js
* 3D & Animations: Three.js (or React Three Fiber)
* Styling: Tailwind CSS or custom CSS
* API Communication: REST API (PHP → React)

---

* Fully responsive
